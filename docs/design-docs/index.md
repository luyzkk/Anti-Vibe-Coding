## ADR Format

Each ADR follows the format:
```
---
title: "..."
status: active | superseded | deprecated
date: YYYY-MM-DD
superseded_by: ADR-NNNN (if applicable)
---

## Context
## Decision
## Alternatives Considered
## Consequences
## Reversibility
```

New ADRs: `ADR-NNNN-kebab-slug.md` where NNNN increments from 0001.
