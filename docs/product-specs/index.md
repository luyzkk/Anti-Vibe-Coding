# Product Specs

Active product specs live here when planning is in progress.

## v6.0.0

The v6.0.0 spec lives in the parent repository at `V6.0.0/` (relative to the monorepo root).
Execution plans for v6.0.0 are in `docs/exec-plans/completed/` after completion.

## Spec Lifecycle

1. PRD created via `/anti-vibe-coding:write-prd` → `docs/exec-plans/active/{date-slug}/PRD.md`
2. Plans generated via `/anti-vibe-coding:plan-feature`
3. After completion, exec-plan archived to `docs/exec-plans/completed/`
4. Product spec index updated here with link to archived plan
