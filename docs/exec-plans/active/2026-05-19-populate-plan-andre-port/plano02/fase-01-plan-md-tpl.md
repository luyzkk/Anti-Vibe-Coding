- **G3 do plano (10 base = `EXEC_PLAN_SECTIONS_FULL`):** ordem case-sensitive. Conferir

  fase-04 o teste E2E rejeita.

  NAO escrever `## Follow-up Plans` com corpo vazio (parity test fase-04 rejeita).

  tpl (ex: em "Como executar"). Se necessario futuramente, documentar nova convencao.
- **Local (frontmatter aspas):** `title` deve estar entre aspas duplas — o template usa
  `"Populate Harness — {{PROJECT_NAME}}"` para evitar YAML quebrar com travessao. Espelha o
  padrao de `renderExecPlan` em `skills/lib/exec-plan-template.ts:39-47` (que usa
  `JSON.stringify` para o title).
- **Local (PHASES_TABLE injection ponto):** o marcador aparece DENTRO de `## Execution Steps`,
